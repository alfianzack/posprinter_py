Contoh Integrasi POS Printer dari Website

Cara Menggunakan

Website ini dapat menggunakan fungsi print dan cash drawer melalui JavaScript bridge yang disediakan oleh aplikasi POS Printer.

1. Print Receipt

2. Buka Cash Drawer

Contoh Kode JavaScript


// Print receipt
if (window.posPrinter) {
    window.posPrinter.print('TOKO CONTOH\nJl. Contoh No. 123\n\nTerima Kasih', true);
}

// Buka cash drawer (default pin)
if (window.posPrinter) {
    window.posPrinter.openCashDrawer();
}

// Buka cash drawer dengan pin spesifik
if (window.posPrinter) {
    window.posPrinter.openCashDrawer(2); // Pin 2
}
